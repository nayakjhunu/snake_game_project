# 🐍 Snake Game (Python)

## 📌 Project Description
This is a simple Snake Game developed using Python. The player controls the snake to eat food and grow longer while avoiding collisions.

---

## 🎮 Features
- Snake movement using keyboard
- Random food generation
- Score tracking system
- Game over when snake hits wall or itself

---

## 🛠️ Technologies Used
- Python
